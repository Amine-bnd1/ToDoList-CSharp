# ToDoList-CSharp

A simple Windows Forms to-do list app built with C# and .NET Framework 4.7.2.

## Features

- Add tasks
- Remove selected tasks
- Import tasks from a `.txt` file
- Export tasks to a `.txt` file

## Tech Stack

- C#
- .NET Framework 4.7.2
- Windows Forms

## Project Structure

```text
ToDoList-CSharp/
├── Form1.cs
├── Form1.Designer.cs
├── Form1.resx
├── Program.cs
├── App.config
├── ToDoListApp.csproj
└── Properties/
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Amine-bnd1/ToDoList-CSharp.git
   ```

2. Open the project in Visual Studio

3. Build and run the app

## Notes

This project is a beginner-friendly example of:
- Windows Forms development
- Event-driven programming
- File handling in C#
